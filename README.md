# タイトル
TECH HAZARD

# 概要
オブジェクト指向によるプログラミングで制作したゲームです。<br>
コマンド選択式のRPGの戦闘のようなシステムです<br>
このゲームには以下のような特徴があります。<br>
・主人公選択<br>
・ナイフと銃による攻撃<br>
・HP回復アイテムの使用<br>
・獲得したポイントにより称号が得られる<br>
